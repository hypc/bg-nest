# BackGround Nest

## Usage

```html
<script src="bg-nest.min.js"></script>
<script>
    new BgNest().draw();
</script>
```

## Options

#### zindex

Type: `int` | `string`

Default: `-1`

the index of z space

#### opacity

Type: `float`

Default: `0.5`

the opacity of line (0~1)

#### color

Type: `string`

Default: `0,0,0`

the canvas line color, the color is (R,G,B)

#### count

Type: `int`

Default: 150

the number of lines
